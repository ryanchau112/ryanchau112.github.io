n8n Job Automation & Scoring Workflow - 2026

Built an automated workflow using n8n to fetch job listings via API, process the data, and store structured results in Google Sheets. The system handles filtering, formatting, and error cases such as empty responses.

Additional logic was implemented to score and prioritise jobs based on relevance, enabling faster decision-making and reducing manual effort. The workflow runs on a self-hosted environment with Docker, ensuring flexibility and full control over integrations.

n8n Automation API Integration Docker Google Sheets

Python Flask CCTV Monitoring Dashboard - 2023-2026

Built a Python Flask application that monitors CCTV device status using IMAP email notifications. The system reads device emails, tracks when devices go offline or come back online, and presents the information in a browser-based dashboard.

The dashboard provides real-time status visibility, downloadable records, automatic updates and IIS integration for access through a Windows server.

Python Flask IMAP IIS HTML

Card Tracker App - in 2022 (before AI)

Built a card-tracking app for the strategic card game. The idea came from a practical gameplay problem: players benefit from tracking previously played cards to better predict future draws.

I designed and built the app from scratch, learning SwiftUI and Xcode, developing a working prototype, preparing supporting documentation, conducting beta testing, and managing the App Store submission process. However, the app was ultimately not permitted under Apple’s App Store guidelines, which restricted the core functionality of the product, leading to the project being discontinued.
